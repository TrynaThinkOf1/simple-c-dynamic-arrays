To use, simple download the .h files above and put them into the same directory.

```
.
├── include
│   ├── dynamic_arrays
│   │   ├── 2DArrays.h
│   │   ├── IntArray.h
│	│	├── DoubleArray.h
│   │   └── String.h
│   ├── other
│   │   └── file.h
│   └── utils.h
└── src
    ├── file.c
    └── main.c
```
