To use, simply download any or all of the .h files inside of headers/ and add them to your project.
They all only depend on the C standard library.

```
.
├── include
│   ├── dynamic_arrays
│   │   ├── 2DArrays.h
│   │   ├── IntArray.h
│	│	├── DoubleArray.h
│   │   └── String.h
│   ├── other
│   │   └── file.h
│   └── utils.h
└── src
    ├── file.c
    └── main.c
```
